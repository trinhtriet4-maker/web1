Trang Kiểm Tra Tài Nguyên

Trang này dùng để kiểm tra việc liên kết các tệp CSS và JavaScript bên ngoài.

Các bước kiểm tra khi CSS hoặc JS không hoạt động

  1. Kiểm tra đường dẫn trong thuộc tính href hoặc src.
  2. Kiểm tra tên tệp và thư mục có đúng hay không.
  3. Mở trình duyệt và nhấn F12 để mở DevTools.
  4. Vào tab Console để xem thông báo lỗi.
  5. Vào tab Network để kiểm tra tệp CSS hoặc JS có được tải hay không.

Nếu xuất hiện lỗi 404 (Not Found), hãy kiểm tra lại đường dẫn đến tệp CSS hoặc JavaScript.
